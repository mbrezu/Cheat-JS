About Cheat-JS

Cheat-JS is a JavaScript preprocessor that permits applying AST transformations on JavaScript source. The AST transformations (macros) are written in Common Lisp. Instead of writing:

var greeter = (function () {
    return { 
        'hello': function(name) {
            console.log('hello, '+ name);
        };
    };
}());

Cheat-JS makes it possible to write:

var greeter = @iife(
    return {
        'hello': function(name) {
            console.log('hello, '+ name);            
        }
    };
);

Instead of writing:

function Person(name, shoeSize) {
    this.name = name;
    this.shoeSize = shoeSize;
}

Cheat-JS makes it possible to write:

@defclass(Person, name, shoeSize)

(assuming that @iife and @defclass were defined as AST transformations in Common Lisp code).
